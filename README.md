### What I learned

I learned to hide and to show image for diferent screen size. in screen mobile change display property:

``` css
.desktop-image {
  display: none
}
```
In media queris, i reverse arguments.

``` css
@media (min-width:375px){
.desktop-image {
  display: block;
}
.mobile-image {
  display: none;
}
}

```
